## LoRa_Gateway


### Configuration

- Change the BOARD definition in `board.h` according to the your board Version " 1 = ON / 0 = OFF ".
    ```
    #define LORA_V1_0_OLED  0
    #define LORA_V1_2_OLED  0
    #define LORA_V1_6_OLED  0
    #define LORA_V2_0_OLED  1
    ```
